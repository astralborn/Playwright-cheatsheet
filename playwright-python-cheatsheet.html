Preferred get_by_* locators

★ Best Practice: Always prefer get_by_test_id() first — it's immune to UI text/structure changes. Add data-testid attributes to your app's elements.
PriorityMethodUse when
1stget_by_test_id("submit-btn")Dedicated data-testid attr — most stable
2ndget_by_role("button", name="OK")ARIA role + name — mirrors user perception
2ndget_by_label("Email")Form fields with <label>
3rdget_by_placeholder("Search")Input placeholder text
3rdget_by_text("Submit")Visible text content
3rdget_by_alt_text("Logo")Images by alt attribute
3rdget_by_title("Close")Title attribute
Avoid CSS selectors, XPath, and .nth() as primary locators — they break on DOM changes. Use them only as a last resort.

Low-level mouse API

Use when the app uses dragover events — requires two mouse moves over the target to trigger it reliably.
# manual mouse-down → move → up page.mouse.move(100, 200) # hover source page.mouse.down() page.mouse.move(500, 200) # move to target (1st) page.mouse.move(500, 200) # repeat for dragover page.mouse.up() # useful for canvas / custom sortables page.mouse.move(100, 100) page.mouse.down() for step in range(0, 400, 10): page.mouse.move(step, 100) page.mouse.up()

Custom soft-assertion collector

from playwright.sync_api import expect # soft assertions continue after failure def test_dashboard(page): page.goto("/dashboard") # these won't stop the test expect.soft(page.get_by_test_id("title")).to_have_text("Dashboard") expect.soft(page.get_by_test_id("user-menu")).to_be_visible() expect.soft(page.get_by_test_id("logout-btn")).to_be_enabled() # hard assertion at the end flushes soft failures expect(page).to_have_url("/dashboard")
Pattern: Use soft assertions for independent checks in one test, then a single hard assertion to fail the test if any soft one failed.

Thread safety note

Important: Playwright's API is not thread-safe. For multi-threaded use, create one playwright instance per thread.
from threading import Thread from playwright.sync_api import sync_playwright def run_in_thread(url): with sync_playwright() as p: # own instance browser = p.chromium.launch() page = browser.new_page() page.goto(url) browser.close() threads = [Thread(target=run_in_thread, args=(u,)) for u in ["/a", "/b"]] for t in threads: t.start() for t in threads: t.join()
